## 第一部分：成都市市级政策

### 一、科技金融资助

| 项目 | 内容 |
|------|------|
| **政策名称** | 《成都市科技金融资助管理办法》 |
| **发布部门** | 成都市科学技术局、成都市财政局 |
| **发布时间** | 2026年4月13日 |
| **有效期** | 5年 |
| **来源** | http://cdst.chengdu.gov.cn/gkml/xzgfxwj/1494346770279301120.shtml |

#### 补贴明细

| 资助类别 | 具体项目 | 补贴标准 |
|---------|---------|---------|
| **天使投资补助** | 获得天使投资的种子期、初创期科技型企业 | 按所获投资额的10%，最高100万元一次性补助 |
| **债权融资 - 贷款利息** | "科创贷"（一般科技型企业） | 按同期LPR的30%，每户每年最高20万元 |
| **债权融资 - 贷款利息** | "科创贷"（人工智能OPC） | 按同期LPR的100%，每户每年最高20万元 |
| **债权融资 - 担保费** | 贷款担保费用 | 按实际发生担保费的50%，每户每年最高20万元 |
| **债权融资 - 债券利息** | 科技创新债券融资利息 | 按同期LPR的30%，每户每年最高20万元 |
| **债权融资 - 债券担保费** | 科技创新债券担保费 | 按实际发生担保费的50%，每户每年最高20万元 |
| **科技保险（重点引导类）** | 研发攻关类、成果转化类、投融资类等 | 按实际支出保费的70%，每家每年最高20万元 |
| **科技保险（一般类）** | 企业责任险、财产险等 | 按实际支出保费的10%，每家每年最高20万元 |

#### 天使投资补助申请条件

- 企业在成都市持续开展经营活动
- 接受投资时从业人数不超过200人（本科以上学历不低于30%），资产总额和年销售收入均不超过3000万元
- 接受投资时企业设立时间不超过5年（60个月）
- 未在境内外证券交易所上市
- 申报时上一年度研发费用占成本费用支出比例不低于20%
- 自天使投资到账满一年后的12个月内提出申请

#### 申请流程

1. 登录"成都市科技项目管理系统"注册并获取申报资格
2. 填写申报书，提交所在区（市）县科技管理部门审核
3. 市科技局窗口审核后，由成都生产力促进中心初审、主管处室复审
4. 社会公示无异议后，由成都生产力促进中心拨付经费

### 二、高新技术企业认定奖补

| 项目 | 内容 |
|------|------|
| **政策名称** | 《成都市高新技术企业认定奖补管理办法》 |
| **发布部门** | 成都市科学技术局、成都市财政局 |
| **文号** | 成科字〔2020〕23号（已更新为"免申即享"模式） |

#### 补贴标准

| 类型 | 市本级奖补 | 说明 |
|------|-----------|------|
| 首次认定 | 15万元（2025年最新"免申即享"标准为10万元） | 区县另有配套，合计最高可达30-40万元 |
| 重新认定 | 5万元 | 区县另有配套 |
| 外地迁入 | 视为首次认定 | 享受同等奖补 |

#### 申请条件

- 在成都市行政区域内注册登记且纳税关系在成都市
- 通过高新技术企业认定
- 按要求完成科技部门火炬报统及核准

#### 申请流程（2025年"免申即享"模式）

1. 市科技局发布高新技术企业认定奖补申报通知
2. 企业登录"成都市科技项目申报系统"确认意愿
3. 公示无异议后直接拨付资金（无需提交纸质材料）

## 第六部分：政策申请通用指南

### 一、政策叠加规则

1. **市级 + 区级可叠加**：高企认定奖补等政策，市级和区级可同时享受
2. **就高不重复**：同一类别补助一般不重复享受，按"就高不重复"原则执行
3. **多类别可并行**：企业可同时申请高企奖补、科技创新券、科技金融资助等不同类别

### 二、"免申即享"改革

成都市正在推行"免申即享"模式，以下政策已支持在线确认即可到账：
- 高新技术企业认定奖补
- 部分区县的高企认定奖补
- 部分区县的研发投入奖励

### 三、常见申请渠道

| 渠道 | 适用政策 | 网址 |
|------|---------|------|
| 成都市科技项目管理系统 | 科技金融资助、高企奖补、创新券等 | cdst.chengdu.gov.cn |
| 成都高新区官网政策申报系统 | 高新区所有政策 | www.cdht.gov.cn |
| 天府新区官网 | 天府新区所有政策 | www.cdtf.gov.cn |
| 各区县就业部门 | 创业补贴、创业担保贷款 | 各区县人社局 |
| "蓉易办"平台 | 综合政务服务 | cds.sczwfw.gov.cn |

### 四、申请材料通用清单

1. 企业营业执照副本
2. 税务登记证明
3. 财务报表（上一年度及最近一期）
4. 研发费用专项审计报告（如适用）
5. 高新技术企业证书（如适用）
6. 知识产权证书（如适用）
7. 人员社保缴纳证明
8. 项目申报书（按模板填写）
9. 其他专项材料（按具体政策要求）

### 五、申请时间节点

- **集中申报期**：大部分政策每年上半年发布申报通知（3-6月）
- **常年申报**：部分政策支持常年申报、分批审核
- **"免申即享"**：认定后自动触发，无需关注申报时间
- **建议**：关注各区县科技局官网获取最新申报通知

## 第七部分：Agent 知识库构建建议

### 一、数据结构化建议

建议将政策数据按以下维度结构化，便于 Agent 检索和匹配：

```
政策条目 {
  policy_id: string          // 唯一标识
  policy_name: string        // 政策名称
  issuing_authority: string  // 发布部门
  region: string[]           // 适用区域（如["成都市","高新区","天府新区"]）
  industry_tags: string[]    // 适用行业标签（如["科技","AI","生物医药"]）
  subsidy_type: string       // 补贴类型（如"认定奖补"、"贷款贴息"、"租金补贴"）
  max_amount: number         // 最高补贴金额（万元）
  conditions: string[]       // 申请条件列表
  required_docs: string[]    // 所需材料列表
  application_process: string[]  // 申请流程步骤
  deadline: string           // 申请截止时间
  validity_period: string    // 政策有效期
  source_url: string         // 政策来源链接
  stacking_rules: string     // 叠加规则说明
}
```

### 二、智能匹配逻辑建议

Agent 应根据用户输入的以下信息进行政策匹配：

1. **企业基本信息**：注册区域、成立时间、营收规模、员工人数
2. **资质信息**：是否高企、是否专精特新、是否科技型中小企业
3. **行业信息**：所属行业、技术领域
4. **需求信息**：需要哪类补贴（资金、场地、贷款、人才等）
5. **发展阶段**：种子期、初创期、成长期、成熟期

### 三、政策更新机制建议

- 建议每月抓取一次各区县政府官网的政策更新
- 重点关注：成都市科技局、成都高新区、天府新区官网
- 设置政策有效期提醒，提前3个月通知用户即将到期的政策

### 四、注意事项

1. **政策时效性**：本报告数据截至2026年5月，具体政策可能存在更新调整
2. **预算管理**：所有补贴政策均实行预算管理和总量控制，实际可申请金额以当年申报通知为准
3. **信息核实**：建议在正式申请前通过官方渠道确认最新版本
4. **官方渠道**：
   - 成都市科技局：cdst.chengdu.gov.cn
   - 成都高新区：www.cdht.gov.cn
   - 四川天府新区：www.cdtf.gov.cn
   - 成都市人社局：cdhrss.chengdu.gov.cn
   - 成都市经信局：jxj.chengdu.gov.cn
